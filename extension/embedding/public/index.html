Embedding jsreport

This is the prototype for jsreport end user customization feature. You can go through three scenarios that came to my mind jsreport could support.

Source code - https://github.com/pofider/jsreport/tree/embedding
Issue and discussion - https://github.com/jsreport/jsreport/issues/20

  1. Customization with input data
  2. Customization with loading data via script
  3. Client side html widgets
